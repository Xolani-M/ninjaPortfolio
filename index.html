Docker-Practice

docker-practice is a basic Node.js web application that showcases how to containerize a Node.js application using Docker. It provides a simple API built with Express, a predefined Dockerfile and a readme.md file with step by step instructions on how to create a Docker image for the application. I created this project for people who are interested in docker and want to know how it works.

I have expanded the docker-practice project's horizons by setting up a simple CI/CD pipeline on GitLab. My pipeline includes build and test stages, ensuring that everything runs smoothly.
